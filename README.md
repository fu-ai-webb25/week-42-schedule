# Schema, vecka 42
###### Utveckling av AI-drivna applikationer, vecka 2 av 4

## Introduktion

Nu när vi kan kommunicera med en språkmodell är nästa steg att styra den på ett effektivt sätt. Den här veckan fokuserar vi på prompt engineering och hur olika tekniker kan användas för att skapa mer förutsägbara och användbara AI-applikationer.

Vi fördjupar oss även i LangChain och lär oss hur flera steg kan kombineras till AI-flöden genom kedjor, verktyg och strukturerade arbetsprocesser.

## Mål för veckan:

1. Förstå hur prompts påverkar språkmodellens svar.
2. Skriva effektiva prompts med olika prompting-tekniker.
3. Använda System- och User-prompts.
4. Arbeta med Prompt Templates.
5. Bygga AI-flöden med flera steg i LangChain.
6. Förstå hur verktyg kan utöka språkmodellens funktionalitet.
